# Implementation Plan - Debug Generation Pipeline

## Problem Description
The roof tile visualizer has two issues during generation:
1. **The entire roof is not selected/changed**: The upper roof remains untouched while only the lower roof above the garage is modified.
2. **Incorrect/flat roof materials**: Even when "terracotta red" is selected, the unchanged roof segments display the original shingles (weathered wood or slate grey). The modified segments appear as flat, solid color patches with no depth or texture.

## Analysis & Root Cause
1. **SAM 3.1 Max Detections**: 
   - ComfyUI's native `SAM3_Detect` node uses a text prompt to perform segmentation. When it receives a standard CLIP text conditioning for `"roof"`, the underlying `sam3_clip.py` wrapper defaults to `max_detections = 1`.
   - Because the roof of the house has multiple disconnected segments (upper roof, garage roof, etc.), a limit of 1 detection forces SAM to output a mask containing *only* the single highest-scoring segment (the garage roof). The upper roof is completely excluded.
   - **Solution**: Change the SAM prompt text from `"roof"` to `"roof:10"` in `app/api/generate/route.ts` and `tests/test_segmentation_api.py`. The `:10` suffix tells the SAM CLIP wrapper to request up to 10 detections, which will capture all separate roof segments and output a complete union mask.

2. **ControlNet Depth Strength Too High**:
   - The depth ControlNet strength is currently set to `0.8`.
   - When strength is `0.8`, the ControlNet heavily forces the generation to match the exact fine, flat depth geometry of the original shingles in the input image. When trying to generate wavy terracotta barrel tiles on top of a flat shingle depth, the model compromises by producing a flat, textureless red-brown surface.
   - **Solution**: Reduce the ControlNet Depth strength to `0.45` (or similar). This gives the generator enough local flexibility to render the 3D depth and wavy geometry of terracotta barrel tiles (or other styles like cedar shakes), while still preserving the overall shape and perspective boundaries of the roof.
