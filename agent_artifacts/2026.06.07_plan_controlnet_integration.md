# Implementation Plan - Integrate ControlNet Depth SDXL (Finalized)

This plan outlines the changes needed to integrate a ControlNet Depth SDXL node into our ComfyUI API workflow. This will preserve the precise layout, shape, and structure of the original roof while allowing the SDXL inpainting model to replace its texture and styling.

## Proposed Changes

### Next.js Backend API

Update the ComfyUI API workflow JSON to:
1. Load the ControlNet Depth SDXL model using `ControlNetLoader` (Node 15).
2. Generate a depth map of the original image using `MiDaS-DepthMapPreprocessor` (Node 16).
