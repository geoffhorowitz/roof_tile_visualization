# Extract Configs to Centralized Config File

Extract hardcoded configuration settings (ComfyUI server URL, model names, KSampler parameters, SAM parameters, controlnet parameters, Supabase storage bucket, local paths, and positive/negative prompts) from `tests/run_real_inpainting.py`, `app/api/generate/route.ts`, and `app/api/image/route.ts` into `config/advanced_user_configs.json`.

## User Review Required

> [!NOTE]
> We will create a unified JSON config file `config/advanced_user_configs.json` containing detailed keys, values, and description metadata fields for all configuration variables.
>
> We will also build:
> 1. A Python helper `utils/config_loader.py` to flatten and load these configurations for scripts/tests.
> 2. A TypeScript helper `utils/configLoader.ts` to load these configurations for the Next.js backend proxy.
