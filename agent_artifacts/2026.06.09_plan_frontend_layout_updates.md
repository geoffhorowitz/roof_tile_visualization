# Frontend Layout and History Features Upgrades

Update the frontend controls, results area, history tracking, and naming conventions.

## User Review Required

> [!NOTE]
> All requested changes will be fully supported in both Supabase mode and Local Mock Mode.
>
> 1. **Default Image**: `data/house.jpg` will be duplicated to `public/house.jpg` so it is served statically on the client.
> 2. **Layout Changes**:
>    - Sidebar panels "Upload Photo" and "Select Roof Style" will be merged into a single glass panel with two tabs.
>    - Main "Result" panel will be converted into a two-tab view: "Render Result" and "Render History".
> 3. **Optional Slider**: The image slider in "Render Result" will be toggleable. When off, the user can toggle "Show Before Image" to swap between before/after views.
> 4. **File Names & Labels**: Generated output files will be stored in Supabase or local output folder using the format `{original_image_name_without_extension}_{tile style}_{tile name}_{timestamp}.png`, and this filename will be extracted and displayed as the item label in the history tab.
